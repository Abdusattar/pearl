## Полный список модулей (roadmap)

### Дети / семьи
- Карточка ребёнка (ФИО, PIN, класс/группа, контакты родителей)
- Посещаемость
- Питание (меню, учёт по дням)
- Развоз (маршруты, расписание)
- Медицина (аллергии, группа здоровья)

### Финансы
- Оплаты (банк → выписка → привязка к ребёнку)
- Расходы (фото квитанций → OCR → категории) ← **реализовано**
- Зарплаты сотрудников / табель
- Отчёты для владельцев

### Учебный процесс
- Учебные планы, оценки, расписание, кружки

### CRM / Коммуникация
- Уведомления родителям (Telegram бот)
- Объявления по классам, договоры

### Операционка
- Сотрудники, инвентарь, поставщики, мероприятия
