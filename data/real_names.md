## Ключевые наблюдения для normalize.py

1. **Кыргызские слова встречаются реально:** `сабиз` = морковь, `пияз` = лук — в каталог нужны как алиасы
2. **Бренды вместо товара:** `Береке` = соль/мука, `Береке соль` = соль, `Айлана` = кефир, `Камолино` = рис
3. **Части курицы по отдельности** — один поставщик даёт чек из 10 позиций: Бедро / Голень / Желудки / Крылья / Окорочок / Печень / Сердце / Тушка / Фарш / Филе
4. **Строчные и заглавные вперемешку** — `морковь` и `Морковь`, `укроп` и `Укроп`
5. **Масло без уточнения** — просто `Масло`, не `Масло растительное`
